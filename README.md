# Vaishnavi Mendon J - Portfolio Website

A stunning, animated portfolio website built with React, TypeScript, and Tailwind CSS featuring a purple/magenta/pink gradient theme on a pure black background.

## 🌟 Features

### Visual Effects & Animations
- **20+ Moving Animated Elements**: Floating particles, geometric shapes, glowing orbs, animated lines, orbiting elements
- **Smooth Scrolling Effects**: Scale, fade, and glow animations between sections
- **Mouse Parallax Effects**: Background elements respond to mouse movement
- **Animated Background**: Mesh gradients, scanlines, and dot patterns
- **Multiple Animation Layers**: Pulse, spin, float, orbit, glow, fade, scale, rotate
- **SVG Line Drawing Animations**: Animated path drawing effects
- **Corner Accent Animations**: Decorative corner elements
- **Glowing Pulse Orbs**: Expanding pulse effects

### Sections
- **Hero**: Animated gradient avatar with floating particles and social links
- **About**: Professional summary with animated feature cards
- **Skills**: Interactive skill bars with certificates
- **Projects**: Showcase of 6 featured projects with hover effects
- **Contact**: Contact form with social media integration
- **Footer**: Complete footer with quick links and contact info

### Technical Features
- **Responsive Design**: Optimized for all devices
- **Custom Gradient Scrollbar**: Themed scrollbar design
- **Section Entrance Effects**: Glow overlays and smooth transitions
- **TypeScript**: Full type safety throughout the application
- **Framer Motion**: Advanced animations and transitions
- **Tailwind CSS**: Utility-first styling with custom animations

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/vai15mendon/intern.git
   cd intern
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```
