🔧 METHOD 2: INSTALL GIT (COMMAND LINE)

If you prefer using command line:

STEP 1: Download Git
   👉 Go to: https://git-scm.com/download/win
   👉 Download will start automatically
   👉 Wait for download to complete

STEP 2: Install Git
   👉 Run the installer
   👉 Use all default settings (just click "Next")
   👉 Click "Install"
   👉 Click "Finish"

STEP 3: Restart Terminal
   👉 Close all terminal/command prompt windows
   👉 Open a NEW terminal/command prompt
   👉 Navigate to your project:
      cd C:\Users\ADMIN\Downloads\intteeerrrrrnnnnn

STEP 4: Configure Git (First Time Only)
   git config --global user.name "Vaishnavi Mendon J"
   git config --global user.email "vaishnavimendonj5@gmail.com"

STEP 5: Initialize and Push
   Run these commands one by one:

   git init
   git add .
   git commit -m "Fix: Vercel deployment error - complete portfolio"
   git branch -M main
   git remote add origin https://github.com/vai15mendon/intern.git
   git push -u origin main

STEP 6: Enter Credentials
   When prompted:
   👉 Username: vai15mendon
   👉 Password: [Use Personal Access Token, NOT your GitHub password]
   
   To generate token:
   👉 Go to: https://github.com/settings/tokens
   👉 Click "Generate new token (classic)"
   👉 Select scope: repo (full control)
   👉 Click "Generate token"
   👉 Copy the token and use it as password
